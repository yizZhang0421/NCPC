給a,r,P，r<P，所以r就是餘數</br>
設Q為商，ax=P*Q+r</br>

    a</br>

從0往上找，找出第一個Q使x為正整數即為答案</br>
也就是((P*Q)+r)/a能整除</br>
相較於一個一個找，更快的做法為地回縮小問題</br>
ex:</br>
