給a,r,P，r<P，所以r就是餘數
設Q為商，ax=P*Q+r
x=
 (P*Q+r)
----------
    a

從0往上找，找出第一個Q使x為正整數即為答案
也就是((P*Q)+r)/a能整除
相較於一個一個找，更快的做法為地回縮小問題
ex:
a=2147465536, P=2147483647, r=1
x=(2147483647Q+1)/2147465536 ==> (2147483647Q % 2147465536)+1需要是2147465536的倍數
(2147483647Q % 2147465536)=(2147483647%2147465536)*(Q%2147465536)=2147465536*(Q%2147465536)
所以形成新的子問題：
設2147465536的倍數為(n1)
2147465536*(Q%2147465536)+1=2147465536(n1)
子問題為(2147465536(n1)-1)/2147465536需要整除
不斷遞迴直到分母=1

所以該題遞迴內容為：
