給a,r,P，r<P，所以r就是餘數</br>
設Q為商，ax=P*Q+r</br>
x=</br>
 (P*Q+r)</br>
----------</br>
    a</br>
</br>
需要整除，所以(P*Q)/a的餘數需要為a-r</br>
(P%a)*(Q%a)=a-r</br>
P%a可以得到</br>
所以可以求出Q%a需要為多少</br>
比a的範圍是(0,P)</br>
所以Q就是求出的Q%a</br>
</br>
ex:</br>
[4 3 5]</br>
4x=5Q+3</br>
x=</br>
  5Q+3</br>
 ------</br>
   4</br>
5Q%4=4-3=1</br>
5%4=1，所以Q%4=1，Q=1</br>
求出x=2</br>
</br>

  256Q+2</br>
 --------</br>
    3</br>

256%3=1，所以Q%3=1,Q=1</br>
